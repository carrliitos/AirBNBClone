## Example usage
*On **non-interactive** mode*

```
$ echo "help" | ./console.py
(ConsoleBNB)
Documented commands (type help <topic>):
========================================
EOF  all  create  destroy  help  quit  show  update

```

*On **interactive** mode*

```
$ ./console.py
(ConsoleBNB) help

Documented commands (type help <topic>):
========================================
EOF  all  create  destroy  help  quit  show  update

(ConsoleBNB) quit

```
